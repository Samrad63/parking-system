# Parking System

سیستم ثبت ورود و خروج خودرو در پارکینگ با محاسبه هزینه بر اساس ساعت  
A web-based parking system to track vehicle entry and exit with fee calculation per hour.

---

## توضیح کوتاه / Overview

این پروژه با **Java 21**, **Spring Boot**, **JDBC** و **Oracle DB** ساخته شده است.  
Front-end با **HTML** و **JavaScript** طراحی شده و امکان ثبت ورود و خروج خودرو، نمایش جدول رکوردها و محاسبه هزینه پارکینگ را دارد.  
The front-end uses HTML and JavaScript to allow vehicle entry/exit registration, display records, and calculate parking fees.

---

## ویژگی‌ها / Features

- ثبت ورود خودرو با شماره پلاک و نوع خودرو  
- Prevent duplicate license plate or parking spot entry
- ثبت خروج خودرو با محاسبه مدت زمان پارک و هزینه  
- نمایش جدول رکوردها با هزینه و وضعیت  
- طراحی **RTL** برای زبان فارسی  
- Simple HTML/JS front-end
